# IComic

## 1.介绍

本项目是基于ssm框架整合实现的一个漫画社区

前端页面是套用素材然后进行修改的 

目前该项目仅用于学习 

数据来源于第三方平台，不做于任何商业用途，如有侵权，立刻删除。
